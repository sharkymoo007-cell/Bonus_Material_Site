# AI Campus Roadshow: Attendee Portal & Bonus Materials

An official repository for the **Microsoft Learn Student Ambassador AI Roadshow** sessions. This repository serves as a centralized hub for workshop attendees to register for future events and download exclusive AI learning resources.

---

## 📂 Repository Structure

The repository contains two primary components designed for rapid deployment and easy access:

*   **`index.html`**: A lightweight, standalone frontend landing page used during live roadshows for student check-ins and newsletter signups.
*   **`Bonus_material.zip`**: A comprehensive archive containing workshop slides, AI prompts and prompt engineering document shares.

---

## 🚀 Getting Started

### For Workshop Attendees

#### 1. Event Registration
Access the live signup portal directly via GitHub Pages to register your attendance and stay updated on upcoming cloud credits and workshops:
👉 **`https://sharkymoo007-cell.github.io/Bonus_Material_Site/`**

#### 2. Downloading Bonus Materials
To download the workshop resource package without cloning the codebase:
1. Locate the `Bonus_material.zip` file in the file list above.
2. Click on the file name, then click the **Download raw file** button on the right-hand side.
3. Extract the ZIP file locally to access the presentation decks and exercises.
